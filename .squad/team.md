### Capabilities

**🟢 Good fit — auto-route when enabled:**
- Bug fixes with clear reproduction steps
- Test coverage additions and flaky test fixes
- Lint, format, and dependency maintenance
- Small isolated features with explicit acceptance criteria
- Documentation fixes and scaffold generation

**🟡 Needs review — route with squad oversight:**
- Medium features with clear specs
- Refactors covered by existing tests
- Pattern-following endpoint or module additions

**🔴 Not suitable — route to squad members instead:**
- Architecture and governance decisions
- Multi-system coordination work
- Ambiguous requirements or approval-gated tasks
- Security-critical or performance-critical changes
