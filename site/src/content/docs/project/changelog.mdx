## Why GitHub Releases?

Maintaining release notes in both the repository and GitHub Releases creates drift risk. The GitHub Releases page is the authoritative source because:

- Release artifacts (if any) are attached directly to the release
- Contributors can subscribe to release notifications
- The release timeline is automatically linked to git tags
