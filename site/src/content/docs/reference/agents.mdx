## Agent Scopes

### Estate-Scoped (operate across all LZs)

- **Conductor** — Orchestrates the full estate
- **Assessor** — Brownfield discovery and WAF assessment
- **Warden** — Cross-cutting governance and policy
- **Sentinel** — Monitors all deployed subscriptions
- **Mender** — Remediates violations across the estate

### Per-LZ Scoped (operate on one landing zone)

- **Scribe, Oracle, Artisan** — Requirements through design
- **Strategist, Forge** — Planning and code generation
- **Envoy, Chronicler** — Deployment and documentation
- **Challenger** — Reviews at gate boundaries

## Subagents

Lightweight specialist agents delegated by parent agents for focused tasks:

| Subagent | Parent Agent(s) | Purpose |
|----------|----------------|---------|
| `bicep-validate-subagent` | Forge (Bicep) | `bicep lint` + `bicep build` + code review |
| `bicep-whatif-subagent` | Forge (Bicep) | `az deployment what-if` preview |
| `terraform-validate-subagent` | Forge (Terraform) | `terraform fmt` + `terraform validate` + code review |
| `terraform-plan-subagent` | Forge (Terraform) | `terraform plan` preview with drift detection |
| `challenger-review-subagent` | Oracle, Strategist, Forge | Focused adversarial review for specific artifacts |
| `cost-estimate-subagent` | Oracle, Strategist | Azure Pricing MCP lookup — real cost estimates |

Subagents are **not user-invocable** — they are called internally by parent agents
and return structured results (PASS/FAIL/NEEDS_REVISION) with actionable findings.
