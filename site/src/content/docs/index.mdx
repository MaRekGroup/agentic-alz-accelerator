## What Is This?

A multi-agent workflow that transforms Azure Landing Zone requirements into deployed,
governed, and continuously monitored infrastructure — aligned with the
Cloud Adoption Framework (CAF) design areas.

  <Card title="12 Specialized Agents" icon="puzzle">

    Monitoring, and Remediation — each with defined responsibilities and artifact handoffs.
    Supports both greenfield and brownfield scenarios.
  </Card>
  <Card title="AVM-First IaC" icon="setting">
    Bicep and Terraform modules built on Azure Verified Modules (AVM).

  </Card>
  <Card title="6 Approval Gates" icon="approve-check">
