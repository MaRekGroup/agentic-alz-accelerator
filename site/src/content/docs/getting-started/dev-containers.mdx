The repository includes a Dev Container configuration with all tools pre-installed.

## Prerequisites

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) or compatible runtime
- [VS Code](https://code.visualstudio.com/) with the [Dev Containers extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)

## Open in Dev Container

1. Clone the repository
2. Open in VS Code
3. Press `F1` → **Dev Containers: Reopen in Container**

The container includes:

| Tool | Purpose |
|------|---------|
| Azure CLI (`az`) | Azure resource management |
| Bicep CLI | Bicep template compilation and linting |
| Terraform CLI | Terraform plan/apply |
| GitHub CLI (`gh`) | Workflow triggering, PR management |
| Python 3 + pip | Validators, MCP servers, diagram generation |
| Node.js + npm | Optional validation scripts |

## Post-Setup

```bash
# Activate Python virtual environment
source .venv/bin/activate
pip install -r requirements.txt

# Authenticate Azure CLI
az login --use-device-code

# Authenticate GitHub CLI
gh auth login
```

## GitHub Copilot Agents

The Dev Container is configured with GitHub Copilot agent definitions in `.github/agents/`.
Use `@workspace` in Copilot Chat to invoke the orchestrator or individual agents.
